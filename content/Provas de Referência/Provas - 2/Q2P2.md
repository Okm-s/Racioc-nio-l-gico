As placas de identificação de um veículo servem para dar uma identidade única ao carro, permitindo seu rastreamento e consulta pelas autoridades.

Considere o padrão de placa Mercosul formado pela sequência:

$L_1L_2L_3N_1L_4N_2N_3$

Em que:

- $L$ representa letra;
    
- $N$ representa algarismo.
    

![[Pasted image 20260620194832.png]]

Considere que todas as letras do alfabeto e todos os algarismos de 0 a 9 podem ser usados, com repetição.

Sabendo que:

- a primeira letra $L_1$ deve ser **A**;
    
- a quarta letra $L_4$ deve ser **B**;
    
- os dois últimos algarismos $N_2$ e $N_3$ devem ser **1** e **5**, respectivamente;
    

assinale quantas combinações diferentes podem ser formadas para uma placa.

a) 456976000 combinações diferentes  
b) 411278400 combinações diferentes  
c) 6760 combinações diferentes  
d) 1340 combinações diferentes  
e) 530 combinações diferentes  


Nessa questão  vamos usar o [[Princípio fundamental da contagem]]

Abaixo temos a configuração da placa. 

| L1  | L2  | L3  | N1  | L4  | N2  | N3  |
| :-: | :-: | :-: | :-: | :-: | :-: | :-: |
|     |     |     |     |     |     |     |
Agora vamos ver quais são as regras que temos que obedecer.

| L1  | L2  | L3  | N1  | L4  | N2  | N3  |
| :-: | :-: | :-: | :-: | :-: | :-: | :-: |
|  A  |     |     |     |  B  |  1  |  5  |
Pronto. observe que, como esses valores já estão fixados, eles são a nossa única opção. ou seja, cada um deles é uma única possibilidade.

| L1  | L2  | L3  | N1  | L4  | N2  | N3  |
| :-: | :-: | :-: | :-: | :-: | :-: | :-: |
|  1  |     |     |     |  1  |  1  |  1  |
Não confunda uma possibilidade com o número 1.

Agora, vamos ver as opções para as casas restantes. no nosso alfabeto temos 26 letras e 10 algarismos. Se podemos repetir tanto letras como números, podemos usar todas as opções nas casas restantes.

| L1  | L2  | L3  | N1  | L4  | N2  | N3  |
| :-: | :-: | :-: | :-: | :-: | :-: | :-: |
|  1  | 26  | 26  | 10  |  1  |  1  |  1  |

Pelo [[Princípio fundamental da contagem]] temos que o total de casos é:

1 x 26 x 26 x 10 x 1 x 1 x 1 = 6.760 combinações diferentes  

Observação: Cuidado com a palavra [[Combinação]] ela pode te induzir ao erro.

> **Conteúdo usado:** [[Princípio fundamental da contagem]] — conteúdo de *Números*, **fora** da apostila de lógica proposicional.
