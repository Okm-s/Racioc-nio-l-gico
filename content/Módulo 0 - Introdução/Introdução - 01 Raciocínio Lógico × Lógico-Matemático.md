## Dois nomes, duas coisas

|            | **Raciocínio Lógico**                                       | **Raciocínio Lógico-Matemático**                          |
| ---------- | ---------------------------------------------------------- | --------------------------------------------------------- |
| O que é    | tirar conclusões corretas pela **estrutura** do pensamento | a lógica **somada** a tópicos de matemática               |
| Disciplina | **Lógica** (forma, validade, argumento)                    | Lógica **+** Aritmética, Conjuntos, Contagem, Sequências… |
| Ferramenta | forma e validade                                           | cálculo, propriedade, contagem                            |

## Pratique

**Questão.** Maria diz: "Para resolver essa questão, preciso saber se a conclusão decorre das premissas — não fazer nenhuma conta." A questão de Maria pertence a qual área?

a) Lógica pura  
b) Aritmética  
c) Conjuntos  
d) Combinatória  
e) Sequências

> [!success]- Gabarito  
> **Letra a.** A pergunta central de Maria é "a conclusão decorre?" — exatamente o critério que separa lógica pura das demais áreas do Raciocínio Lógico-Matemático.
