# Apostila de Conjuntos

> Índice (MOC). Voltar ao mapa geral: [[Raciocínio Lógico]]

A **teoria dos conjuntos** é a linguagem que descreve agrupamentos de objetos. Serve de base para contagem, probabilidade e para os problemas de Venn que caem em prova.

## Roteiro
1. [[Conjuntos - 01 Notação, Pertinência e Representação]] — como escrever conjuntos; $\in$ e $\notin$.
2. [[Conjuntos - 02 Tipos de Conjuntos]] — vazio, unitário, finito, infinito, universo.
3. [[Conjuntos - 03 Inclusão, Igualdade e Partes]] — $\subset$, $=$, subconjuntos, $P(A)$.
4. [[Conjuntos - 04 Operações com Conjuntos]] — união, interseção, diferença, complementar.
5. [[Conjuntos - 05 Problemas com Diagramas de Venn]] — contagem por regiões (inclusão–exclusão).
6. [[Conjuntos - 06 Banco de Questões]] — treino com gabarito.

## Símbolos-chave

| Símbolo | Significado |
| :-: | --- |
| $\in$ / $\notin$ | pertence / não pertence |
| $\subset$ / $\subseteq$ | está contido (subconjunto) |
| $\supset$ | contém |
| $\cup$ | união |
| $\cap$ | interseção |
| $A - B$ | diferença |
| $\overline{A}$ / $A^c$ | complementar |
| $\varnothing$ | conjunto vazio |
| $P(A)$ | conjunto das partes |
| $n(A)$ | número de elementos de $A$ |

> Ligações: a contagem por regiões liga-se à [[Apostila de Análise Combinatória e Probabilidade]]; o número de subconjuntos ($2^n$) usa a mesma ideia das linhas da [[02 - Tabela-verdade|tabela-verdade]].
