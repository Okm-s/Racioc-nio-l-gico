## Potenciação

$$\underbrace{2}_{\text{base}}{}^{\underbrace{3}_{\text{expoente}}} = 2 \times 2 \times 2 = 8 \ (\text{potência})$$

O expoente diz **quantas vezes a base se multiplica por si mesma**.

### Casos especiais

| Caso | Valor | Exemplo |
| --- | :-: | --- |
| Expoente $1$ | a própria base | $7^1 = 7$ |
| Expoente $0$ | $1$ (base $\neq 0$) | $9^0 = 1$ |
| Base $1$ | $1$ | $1^{100} = 1$ |
| Base $10$ | $1$ seguido de $n$ zeros | $10^3 = 1000$ |
| Expoente negativo | inverte: $a^{-n} = \frac{1}{a^n}$ | $2^{-2} = \frac{1}{4}$ |

### Propriedades das potências

- Mesma base, multiplicando: **soma** expoentes → $a^m \cdot a^n = a^{m+n}$
- Mesma base, dividindo: **subtrai** expoentes → $a^m \div a^n = a^{m-n}$
- Potência de potência: **multiplica** expoentes → $(a^m)^n = a^{m\cdot n}$
- Produto elevado: $(a\cdot b)^n = a^n \cdot b^n$

> [!warning] Sinal da base negativa
> $(-2)^2 = 4$ (expoente par → positivo); $(-2)^3 = -8$ (expoente ímpar → negativo).

## Radiciação

É a operação **inversa** da potenciação.
$$\sqrt[n]{a} = b \quad\Longleftrightarrow\quad b^n = a$$
> $\sqrt{25} = 5$ porque $5^2 = 25$. $\sqrt[3]{8} = 2$ porque $2^3 = 8$.

No símbolo $\sqrt[n]{a}$: **n** é o índice, **a** é o radicando. Sem índice, é raiz quadrada ($n=2$).

> [!tip] Raiz é potência de expoente fracionário
> $\sqrt[n]{a} = a^{1/n}$. Ex.: $\sqrt{a} = a^{1/2}$.

## Pratique

**Questão.** Qual o valor de $(-2)^3$?

a) $8$  
b) $-8$  
c) $6$  
d) $-6$  
e) $1$

> [!success]- Gabarito  
> **Letra b.** Expoente ímpar com base negativa dá resultado negativo: $(-2)^3 = -2 \times -2 \times -2 = -8$.
